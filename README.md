- **version** : `2.319.1`
- **배포일** : `2024-09-03`
- https://hehelee.github.io/react-deploy/

 ## 프로젝트 개요
 - **대상 사용자** : 카카오톡을 이용하는 모든 유저
 - **프로젝트 목적** : 카카오테크 캠퍼스 부트캠프에서 진행한 수업으로 FE의 실무적인 지식을 함양하기 위함.

- 로그인 & 회원가입 기능
- 나에게 선물하기 기능

### [추가 기능]
- 상품 추가, 삭제, 수정 기능
- 관심 상품 등록 기능
